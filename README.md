# frontend-boilerplate

Modern frontend boilerplace where start coding.

# Including

- Build
  - `webpack`
  - CircleCI
  - Amazon S3
- Template ... `Pug`
- Style
  - `PostCSS`
  - CSS Framework `Skeleton`
  - `postcss-simple-vars`
- JS
  - `es2015` with `babel`
  - `Vue.js`
  - `axios`

# Install

```
yarn install
yarn run dev
```
